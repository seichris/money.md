# Money skill pack

This folder contains multiple agent skills, each in its own kebab-case folder with a SKILL.md file.

Skills included
- money-decision-tree
- money-evm-payments
- money-evm-custody
- money-solana-payments
- money-solana-custody
- money-receiving-reconciliation

Packaging
- To upload a single skill, zip that skill folder.
- To share the full pack, zip this entire directory.

Note
- Skill folders intentionally do not contain README.md, following the Anthropic guidance.
